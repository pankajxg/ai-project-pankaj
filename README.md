# SmartTrendAI_Project

## Overview
AI-Powered Market Trend Analyzer with a Streamlit web interface and console demo.
This package contains the source code, a sample dataset, and a short project report.

## Run locally
1. Create and activate a virtual environment
2. Install dependencies
   pip install -r requirements.txt
3. Run console demo
   python src/main.py
4. Run the web app
   streamlit run app.py

## Run on Google Colab
1. Upload the ZIP contents to your Colab workspace or mount Google Drive.
2. Install requirements
   !pip install -r requirements.txt
3. Run the demo
   !python src/main.py
4. Optionally run streamlit (requires additional Colab setup)
   !streamlit run app.py

## Files
- src/: python modules
